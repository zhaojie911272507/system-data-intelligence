# system-data-Intelligence

> A professional **System Agent Skill** for system-level file operations, deep data analysis, visualization, database connectivity, and API integration with built-in security and fallback.

**Capabilities:** System File Access (auto-fallback) | Deep Analysis | 8+ Chart Types | Database (SQL/MySQL/PostgreSQL) | REST API | Security & Privacy

**Install:** `pip install -r requirements.txt`
**Docker:** `docker compose up -d`
**Tests:** `pytest tests/ -v`
